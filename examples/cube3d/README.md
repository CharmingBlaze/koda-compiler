# 3D Cube

Orbiting camera with a spinning cube and satellite cubes.

**Uses raw Raylib** — `use raylib;` gives access to **all 548** Raylib functions. This sample calls `InitWindow`, `DrawCube`, `DrawGrid`, `BeginMode3D`, etc. directly (no `koda.game` required).

```powershell
koda run src/main.koda
```

See also: `examples/spinning-cube` (minimal 3D intro), `wrappers/raylib/api_reference.md` (complete API).
