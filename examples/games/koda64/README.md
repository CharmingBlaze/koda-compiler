# Koda64 — Mario 64-style 3D prototype

Third-person platformer using the Raylib shim (no models required). Demonstrates **JavaScript-style dot notation**: `player.x`, `cam.update()`, `coins[i].on`, object methods with `this`.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move (relative to camera) |
| Space | Jump |
| Right mouse drag | Orbit camera |
| Mouse wheel | Zoom in/out |
| R | Restart |

Collect **5 coins**, then touch the **star** on the hill. Avoid the Goomba!

## Run

From this folder in Koda Studio: **Run** (or `koda run`).

Open in Studio:

```powershell
cd path\to\Koda
.\koda-ide\build\bin\Koda Studio.exe examples\games\koda64
```
