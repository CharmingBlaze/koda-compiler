# KODA 64

Third-person platformer using the **full Raylib wrapper** (`use raylib;`). Demonstrates dot notation (`player.x`, `camera.yaw`, `coins[i].on`), struct methods, and `koda.camera` / `koda.input` helpers.

## Run

```bash
cd examples/games/koda64
koda run
```

Controls: **WASD** move, **Space** jump, **R** reset.
