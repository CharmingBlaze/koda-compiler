# Mario 64 — Koda Studio project

Mario 64-style 3D platformer with orbit camera, coins, and a star goal.

## Controls

| Input | Action |
|-------|--------|
| WASD | Move (camera-relative) |
| Space | Jump |
| Right mouse drag | Orbit camera |
| Mouse wheel | Zoom |
| R | Restart |

Collect **5 coins**, climb the hill, grab the **star**.

## Run in Koda Studio

1. Open this folder as workspace in **Koda Studio**
2. Press **F5** (Run)

Or from terminal:

```bash
koda run
```
