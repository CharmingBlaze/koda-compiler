## Building

```text
-Ithird_party/raylib_static/stage/include
```

```powershell
$env:KODA_NATIVE_SOURCES = ".\wrapper.c"
$env:KODA_LINKFLAGS = "-Ithird_party/raylib_static/stage/include"
koda build game.koda -o game.exe
```

```bash
export KODA_NATIVE_SOURCES="$PWD/wrapper.c"
export KODA_LINKFLAGS="-Ithird_party/raylib_static/stage/include"
koda build game.koda -o game
```
